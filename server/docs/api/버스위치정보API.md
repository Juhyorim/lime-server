
# API 링크

---

### 국토교통부_(TAGO)_버스위치정보
- https://www.data.go.kr/data/15098533/openapi.do
- 데이터 수정일: 	2023-07-12

<br/>

# 활용 방안

---

- 지도에 현재 도착예정인 버스의 실시간 위치를 표시할 때 사용할 수 있을 것 같다.
- 노선 위에서 자동차가 실시간으로 움직이는 기능이 생긴다면 이 API를 사용하면 좋겠따


<br/>

# 데이터 내용

---

### 1. 노선별버스위치 목록조회

- request: 도시코드, 노선 ID
- response: 
  - 배열로 주어짐
  - 노선번호
  - 맵매칭 Y 좌표
  - 맵매칭 X 좌표
  - 정류소 순서
  - 정류소명
  - 정류소 ID
  - 노선유형 (간선버스)
  - 차량번호 (대전 99가 9999)


<br/>

### 2. 노선별특정정류소접근 버스위치정보조회

- request: 노선 ID, 정류소 ID, 도시코드
- response: 
  - 배열로 주어짐
  - 노선 번호
  - 정류소명
  - 맵매칭 Y 좌표
  - 맵매칭 X 좌표
  - 노선 유형

<br/>

### 3. 도시코드 목록 조회

그냥 도시 코드, 도시이름 나오는 API
